# Tecnologías Utilizadas
- Python: Lenguaje de programación principal

- Tkinter: Biblioteca para la interfaz gráfica

- Socket: Comunicación en red TCP/IP

- Threading: Manejo de hilos para operaciones concurrentes

- JSON: Formato para intercambio de datos
  
# Funcionalidades Detalladas
## Chat Público
  Envío y recepción de mensajes en tiempo real

  Interfaz diferenciada con banner azul

## Chat Privado
Doble clic en usuarios para iniciar chat privado

Ventanas independientes con banner rojo

Historial de conversación independiente

## Transferencia de Archivos
Selección de archivos mediante diálogos nativos

Envío público (a todos los usuarios) o privado

Sistema de descarga con confirmación

Carpeta de descargas automática

## Gestión de Usuarios
Lista actualizada de usuarios conectados

Notificaciones de entrada y salida de usuarios

Validación de nombres de usuario únicos
